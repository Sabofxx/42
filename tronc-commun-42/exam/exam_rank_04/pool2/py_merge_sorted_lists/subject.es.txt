Nombre del ejercicio: py_merge_sorted_lists
Archivos esperados  : py_merge_sorted_lists.py
Funciones permitidas: Ninguna
--------------------------------------------------------------------------------

Escribe una función que combine múltiples listas ordenadas en una única lista ordenada,
manteniendo el orden de forma eficiente.

Tu función debe declararse de la siguiente manera:

def merge_sorted_lists(lists: list[list[int]]) -> list[int]:

La función debe:
- Recibir una lista de listas de enteros ordenadas como entrada
- Devolver una única lista combinada en orden ascendente
- Conservar todos los elementos duplicados en el resultado final
- Manejar correctamente listas vacías y entradas vacías
- Mantener una eficiencia óptima para entradas de gran tamaño

Reglas:
- Se garantiza que todas las listas de entrada estarán ordenadas de forma ascendente.
- Las listas vacías deben ignorarse durante la combinación
- Devolver una lista vacía si no se proporciona ninguna entrada válida
- Conservar los duplicados entre diferentes listas
- Manejar correctamente los números negativos

Ejemplos:

Entrada: merge_sorted_lists([[1, 3, 5], [2, 4, 6]])
Salida: [1, 2, 3, 4, 5, 6]

Entrada: merge_sorted_lists([[1, 5, 9], [2, 3, 8], [4, 6, 7]])
Salida: [1, 2, 3, 4, 5, 6, 7, 8, 9]

Entrada: merge_sorted_lists([[5], [1, 3], [2, 4]])
Salida: [1, 2, 3, 4, 5]

Entrada: merge_sorted_lists([[1, 1, 2], [2, 3, 3]])
Salida: [1, 1, 2, 2, 3, 3]

Entrada: merge_sorted_lists([[], [1, 2, 3]])
Salida: [1, 2, 3]

Entrada: merge_sorted_lists([])
Salida: []

Entrada: merge_sorted_lists([[-5, -1, 0], [-3, 2, 4]])
Salida: [-5, -3, -1, 0, 2, 4]

Entrada: merge_sorted_lists([[10], [10], [10]])
Salida: [10, 10, 10]

Casos límite a manejar:
- Lista de entrada vacía: devolver lista vacía
- Listas que solo contienen listas vacías: devolver lista vacía
- Entrada con una sola lista: devolver una copia de esa lista
- Todos los elementos duplicados: conservar todas las instancias
- Números negativos: manejar correctamente el orden
