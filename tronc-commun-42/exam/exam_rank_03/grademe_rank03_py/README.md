# 42 Exam Rank 03 Python - Practice Shell

An interactive practice environment for 42 School Python Exam Rank 03.

## Quick Start

```bash
cd grademe_rank03_py
make
```

Or directly:
```bash
bash exam.sh
```

## Features

- **Level Mode**: Choose Level 1 (basics) or Level 2 (advanced)
- **Real Exam Mode**: Random exercises, simulates real exam
- **Auto-tester**: Type `test` to run automated tests on your code
- **Rendu folder**: Write your solutions in the `rendu/` folder

## Exercises

### Level 1 - Basics
| Exercise | Description |
|----------|-------------|
| atoi | Convert string to integer (no int()) |
| alternate_case | Alternate upper/lower case letters |
| capitalize_words | Capitalize first letter of each word |
| valid_palindrome | Check palindrome ignoring case/symbols |
| valid_anagram | Check if two strings are anagrams |
| merge_and_sort_desc | Merge two lists, sort descending |
| sorted | Master sorted() with key=lambda |

### Level 2 - Advanced
| Exercise | Description |
|----------|-------------|
| brackets | Validate matched brackets |
| convert_base | Convert number between bases [2-36] |
| mirror_matrix | Mirror matrix horizontally |
| mirror_matrix_vertical | Mirror matrix vertically |
| rotate_90 | Rotate matrix 90 degrees clockwise |
| whisper_lipher | Caesar cipher |

## Commands

- `test` - Run the tester on your code
- `next` - Skip to next exercise
- `menu` - Return to main menu
- `exit` - Quit

## Requirements

- Python 3
- Bash shell
- macOS or Linux
