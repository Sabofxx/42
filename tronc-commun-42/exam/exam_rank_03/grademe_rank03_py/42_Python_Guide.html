Strings

s.isalpha() essentiel

True si TOUS les chars sont des lettres. False si chiffre, espace, ponctuation.

"abc".isalpha()   # True
"a1b".isalpha()   # False
"a b".isalpha()   # False
s.isdigit()

True si TOUS les chars sont des chiffres 0–9.

"123".isdigit()   # True
"12a".isdigit()   # False
s.isalnum() essentiel

True si lettres OU chiffres. Exclut espaces et ponctuation.

"abc1".isalnum()  # True
"a b".isalnum()   # False
"a!b".isalnum()   # False
s.upper() / s.lower() essentiel

Retournent une NOUVELLE string. Les strings sont immutables.

"hello".upper()   # "HELLO"
"WORLD".lower()   # "world"
s.isupper() / s.islower()

True si tous les chars alphabétiques sont en maj/min. Les non-lettres sont ignorés.

"ABC".isupper()   # True
"A1C".isupper()   # True (1 ignoré)
"Abc".isupper()   # False
s.capitalize()

1ère lettre majuscule, tout le reste minuscule.

"hELLO".capitalize() # "Hello"
"42abc".capitalize() # "42abc"
s.strip() essentiel

Supprime les espaces, \n, \t en début et fin. Essentiel pour atoi.

"  hello  ".strip()  # "hello"
"  42\n".strip()     # "42"
s.split(sep) essentiel

split(" ") préserve les espaces multiples. split() sans arg les supprime.

"a  b".split(" ")  # ["a","","b"]
"a  b".split()    # ["a","b"]
sep.join(liste) essentiel

Assemble une liste en string avec un séparateur.

" ".join(["A","B"])  # "A B"
"".join(["x","y"])   # "xy"
Slicing — s[start:end:step] essentiel

Syntaxe générale. start inclus, end exclu. Step négatif = sens inverse.

s = "Python"
s[0]      # "P"       — premier char
s[-1]     # "n"       — dernier char

Pattern fondamental — construire une string

Les strings sont immutables : on ne peut pas faire s[i] = 'X'. On reconstruit toujours.

result = ""
for c in s:
    if c.isalpha():
        result += c.upper()   # ajouter à la fin
    else:
        result += c
return result

ASCII · ord / chr

ord(c) → valeur ASCII
ord('a')  # 97
ord('z')  # 122
ord('A')  # 65
ord('Z')  # 90
ord('0')  # 48
chr(n) → caractère
chr(97)   # 'a'
chr(65)   # 'A'
chr(48)   # '0'
# Astuce sans int() :
ord(c) - ord('0')  # '7' → 7
Formule César — whisper_lipher essentiel
(ord(char) − base + shift) % 26 + base
# base = ord('A') si char.isupper() sinon ord('a')
# % 26 assure le comportement circulaire (z → a)

# Exemple : 'H', shift=3
# ord('H')=72, base=65
# (72 - 65 + 3) % 26 = 10
# 10 + 65 = 75 → chr(75) = 'K' ✓

# Exemple : 'z', shift=2
# (122 - 97 + 2) % 26 = 1 → chr(98) = 'b' ✓

Stack (pile) essentiel

En Python, une liste utilisée comme pile LIFO.

stack = []
stack.append('(')   # push
stack.append('[')
stack[-1]          # lire le sommet : '['
stack.pop()        # pop : retourne '[' et l'enlève
len(stack) == 0    # stack vide ?
Dict — accès O(1) essentiel
d = {'a': 1, 'b': 2}
d['a']               # 1
d.get('c', 0)        # 0 si absent
'a' in d             # True

# Pattern brackets :
pairs = {')':'(', ']':'[', '}':'{'}
pairs[')']             # '('
Pattern complet — brackets
def brackets(s: str) -> bool:
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}

    for char in s:
        if char in '([{':
            stack.append(char)                # ouvrant → empiler
        elif char in ')]}':
            if not stack or pairs[char] != stack[-1]:
                return False               # pas de correspondant
            stack.pop()
        # autres chars → ignorés

    return len(stack) == 0                   # tout fermé ?
💡  Deux conditions pour False : pile vide (rien à fermer) OU le sommet ne correspond pas.

Alterner majuscule/minuscule sur les lettres. Les non-lettres sont copiés tels quels et ne comptent pas dans l'alternance.

def alternate_case(s: str) -> str:
    result = ""
    upper = True           # commence par majuscule
    for c in s:
        if c.isalpha():
            result += c.upper() if upper else c.lower()
            upper = not upper   # inverser SEULEMENT pour lettres
        else:
            result += c
    return result

def atoi(s: str) -> int:
    s = s.strip()
    sign, i, result = 1, 0, 0
    if i < len(s) and s[i] in '+-':
        sign = -1 if s[i] == '-' else 1
        i += 1
    while i < len(s) and s[i].isdigit():
        result = result * 10 + (ord(s[i]) - ord('0'))
        i += 1
    return sign * result

Convertir un nombre d'une base à une autre (2–36). print() le résultat (ne pas return). "ERROR" si invalide.

def convert_base(number: str, from_base: int, to_base: int) -> None:
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    try:
        if not (2 <= from_base <= 36 and 2 <= to_base <= 36):
            return print("ERROR")
        decimal = int(number, from_base)   # Python gère ça nativement
    except ValueError:
        return print("ERROR")
    if decimal == 0:
        return print("0")
    result = ""
    while decimal > 0:
        result = digits[decimal % to_base] + result  # PREPEND
        decimal //= to_base
    print(result)

whisper_lipher (César)
stringmathmed
▼ 
def whisper_lipher(text: str, shift: int) -> str:
    result = ""
    for char in text:
        if char.isalpha():
            base = ord('A') if char.isupper() else ord('a')
            shifted = (ord(char) - base + shift) % 26 + base
            result += chr(shifted)
        else:
            result += char
    return result
