Assignment name: atoi
Expected files: atoi.py
Allowed functions: None (no int() or built-in conversion)
--------------------------------------------------------------------------------

Write a function atoi(s) that converts a string to an integer without using
int() or any built-in conversion function.

The function must handle:
- Leading whitespace
- An optional sign (+ or -)
- Stop parsing at the first non-digit character

Examples:

atoi("42")        -> 42
atoi("  -42abc")  -> -42
atoi("+123")      -> 123
atoi("abc")       -> 0
