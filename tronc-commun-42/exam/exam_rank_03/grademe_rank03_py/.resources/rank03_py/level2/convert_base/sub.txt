Assignment name: convert_base
Expected files: convert_base.py
Allowed functions: All built-in (no int() with base parameter)
--------------------------------------------------------------------------------

Write a function:

def convert_base(number: str, from_base: int, to_base: int) -> None:

Converts a number represented as a string from base from_base to base to_base.

Requirements:
- Bases are in the range [2, 36].
- The function must print the result (do not return it).
- Print "ERROR" if any error occurs (invalid digit for base, invalid base, etc).
- Digits above 9 use uppercase letters: A=10, B=11, ... Z=35

Examples:

convert_base("ff", 16, 2)   -> prints "11111111"
convert_base("10", 2, 10)   -> prints "2"
convert_base("z", 36, 10)   -> prints "35"
convert_base("1g", 16, 10)  -> prints "ERROR"
