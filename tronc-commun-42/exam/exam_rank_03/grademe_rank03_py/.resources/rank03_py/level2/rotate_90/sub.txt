Assignment name: rotate_90
Expected files: rotate_90.py
Allowed functions: All built-in
--------------------------------------------------------------------------------

Write a function:

def rotate_90(matrix: list[list[int]]) -> list[list[int]]:

Returns a new matrix rotated 90 degrees clockwise.
The original matrix must not be modified.
The input matrix is assumed to be square.

Example:

matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

rotate_90(matrix) ->
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3]
]
