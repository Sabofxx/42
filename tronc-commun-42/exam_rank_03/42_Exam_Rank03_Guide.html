01 alternate_case
string EASY
▼
Consigne

Retourner une nouvelle string où les caractères alphabétiques alternent majuscule/minuscule. Les caractères non-alphabétiques sont copiés tels quels et ne comptent pas dans l'alternance.

Solution
pythondef alternate_case(s: str) -> str:
    result = ""
    upper = True           # bascule : commence par MAJUSCULE

    for c in s:
        if c.isalpha():      # seulement les lettres
            if upper:
                result += c.upper()
            else:
                result += c.lower()
            upper = not upper  # inverser après chaque LETTRE
        else:
            result += c         # chiffre/espace/ponctuation → intact
    return result
Exemples
"hello world"→"HeLlO WoRlD"
"42madrid"→"42MaDrId"
"a!b?c"→"A!b?C"
Points clés
c.isalpha()
Vrai uniquement pour les lettres (pas les chiffres, espaces, ponctuations).
upper = not upper
S'exécute SEULEMENT si isalpha() → les non-lettres ne cassent pas l'alternance.
c.upper() / c.lower()
Méthodes de string intégrées, retournent une nouvelle string.
02 atoi
string math MED
▼
Consigne

Convertir une string en entier sans utiliser int(). Gérer les espaces de début, le signe optionnel (+/-), et s'arrêter au premier caractère non-digit.

Solution
pythondef atoi(s: str) -> int:
    s = s.strip()           # 1. supprimer les espaces
    sign = 1
    i = 0
    result = 0

    # 2. détecter le signe
    if i < len(s) and s[i] in '+-':
        sign = -1 if s[i] == '-' else 1
        i += 1

    # 3. lire les chiffres un par un
    while i < len(s) and s[i].isdigit():
        result = result * 10 + int(s[i])  # décaler + ajouter
        i += 1

    return sign * result
⚠️
Le sujet dit "sans int() ou conversion built-in", mais on utilise int(s[i]) sur un seul caractère. Alternative : result = result * 10 + (ord(s[i]) - ord('0')) si interdit totalement.
Exemples
"42"→42
" -42abc"→-42
"+123"→123
"abc"→0
Astuce mémorisation
result * 10 + digit
"42" → 4 → 4*10+2 = 42. Décaler la valeur vers la gauche à chaque chiffre.
ord(c) - ord('0')
Alternative sans int() : '7' → 55-48 = 7.
03 brackets
stack MED
▼
Consigne

Retourner True si tous les brackets de la string sont correctement appairés et imbriqués. Ignorer les autres caractères.

Solution (Stack classique)
pythondef brackets(s: str) -> bool:
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}

    for char in s:
        if char in '([{':
            stack.append(char)          # ouvrant → empiler
        elif char in ')]}':
            if not stack or pairs[char] != stack[-1]:
                return False           # fermant sans correspondant
            stack.pop()               # dépiler le correspondant

    return len(stack) == 0          # stack vide = tout fermé
Exemples
"()"→True
"([{}])"→True
"(]"→False
"([)"→False
"a(b[c]d)"→True
Points clés
stack[-1]
Lire le sommet de la pile sans dépiler (dernier élément de la liste).
dict pairs
Mapper chaque fermant vers son ouvrant correspondant. Lookup O(1).
len(stack) == 0
Si la pile n'est pas vide à la fin, il reste des ouvrants non fermés.
04 capitalize_words
string EASY
▼
Consigne

Première lettre de chaque mot en majuscule, le reste en minuscule. Préserver les espaces multiples. Les mots sont séparés par des espaces.

Solution courte (avec split)
pythondef capitalize_words(s: str) -> str:
    words = s.split(" ")                    # split sur espace SIMPLE
    capitalized = [w.capitalize() for w in words]
    return " ".join(capitalized)
ℹ️
split(" ") avec un espace simple préserve les espaces multiples (contrairement à split() sans argument qui les supprime).
Solution manuelle (sans capitalize)
pythondef capitalize_words_manual(s: str) -> str:
    result = ""
    new_word = True

    for char in s:
        if char.isalpha():
            if new_word:
                result += char.upper()
                new_word = False
            else:
                result += char.lower()
        else:
            result += char
            if char == " ":
                new_word = True
    return result
Exemples
"hello world"→"Hello World"
"mixed CASE letters"→"Mixed Case Letters"
" multiple spaces "→" Multiple Spaces "
05 convert_base
math algo HARD
▼
Consigne

Convertir un nombre (string) d'une base à une autre (2–36). print() le résultat, ne pas le retourner. Afficher "ERROR" en cas d'erreur.

Solution
pythondef convert_base(number: str, from_base: int, to_base: int) -> None:
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"

    try:
        # Vérifier les bases
        if not (2 <= from_base <= 36 and 2 <= to_base <= 36):
            print("ERROR")
            return
        # 1. Convertir en décimal (Python gère toutes les bases)
        decimal = int(number, from_base)
    except ValueError:
        print("ERROR")   # chiffre invalide pour la base
        return

    if decimal == 0:
        print("0")
        return

    # 2. Convertir du décimal vers to_base
    result = ""
    while decimal > 0:
        result = digits[decimal % to_base] + result  # prepend le chiffre
        decimal //= to_base
    print(result)
Algorithme de conversion (mémoriser)
# Exemple : décimal 255 → binaire
# 255 % 2 = 1  →  "1"
# 127 % 2 = 1  →  "11"
# 63  % 2 = 1  →  "111"
# ...
# On PREPEND à chaque fois → 11111111
Exemples
convert_base("ff", 16, 2)→11111111
convert_base("10", 2, 10)→2
convert_base("1g", 16, 10)→ERROR
06 merge_and_sort_desc
list EASY
▼
Consigne

Fusionner deux listes d'entiers et retourner le résultat trié du plus grand au plus petit. Ne pas modifier les listes originales.

Solution
pythondef merge_and_sort_desc(list1: list[int], list2: list[int]) -> list[int]:
    merged = list1 + list2       # + crée une NOUVELLE liste
    merged.sort(reverse=True)   # tri descendant in-place
    return merged
💡
list1 + list2 crée une nouvelle liste, donc les originales ne sont pas modifiées. Attention : list1.extend(list2) modifierait list1.
Exemples
[1,3,5], [2,4,6]→[6,5,4,3,2,1]
[10,2], [3,7,2]→[10,7,3,2,2]
[], [1,2,3]→[3,2,1]
07 mirror_matrix

▼
Consigne

Retourner une nouvelle matrice où chaque ligne est inversée horizontalement (miroir gauche↔droite).

Solution
pythondef mirror_matrix(matrix: list[list[int]]) -> list[list[int]]:
    return [row[::-1] for row in matrix]
    # row[::-1] = inverser chaque ligne
    # list comprehension = créer une nouvelle liste

09 rotate_90
matrix algo HARD
▼
Consigne

Retourner une nouvelle matrice carrée pivotée de 90° dans le sens horaire.

Solution
pythondef rotate_90(matrix: list) -> list:
    n = len(matrix)
    return [[matrix[n-1-j][i] for j in range(n)] for i in range(n)]
Formule magique à mémoriser
# Rotation 90° horaire :
# new[i][j] = old[n-1-j][i]

# Dérivation :
# Transposer    → new[i][j] = old[j][i]
# Puis inverser → new[i][j] = old[n-1-j][i]  ✓

# Visualisation :
# [1, 2, 3]        [7, 4, 1]
# [4, 5, 6]  →90°→ [8, 5, 2]
# [7, 8, 9]        [9, 6, 3]

# La COLONNE de gauche devient la LIGNE du bas
# La colonne du bas devient la ligne de droite
💡
Mémo : la colonne 0 (7,4,1) devient la ligne 0 (7,4,1). Les colonnes de gauche → lignes du haut.
10 sorted + lambda
lambda algo MED
▼
Concept — pas une fonction unique

Ce n'est pas un seul exercice mais un concept à maîtriser : trier avec sorted() et un critère personnalisé via key=lambda.

Cas 1 — Trier par longueur
python# Tri par longueur croissante
sorted(["banana", "kiwi", "fig"], key=lambda x: len(x))
# → ["fig", "kiwi", "banana"]
Cas 2 — Trier des tuples par un champ
python# Tri par note décroissante (reverse=True)
students = [("Ana", 8.5), ("Luis", 6.0), ("Marta", 9.2)]
sorted(students, key=lambda x: x[1], reverse=True)
# → [("Marta", 9.2), ("Ana", 8.5), ("Luis", 6.0)]
Cas 3 — Critères multiples (tuple de clés)
python# D'abord par âge, puis par nom si âge égal
data = [("Ana", 20), ("Luis", 20), ("Marta", 18)]
sorted(data, key=lambda x: (x[1], x[0]))
# → [("Marta", 18), ("Ana", 20), ("Luis", 20)]
# age 18 < 20, puis "Ana" < "Luis" alphabétiquement
Cas 4 — Trier des objets
pythonclass Student:
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade

students = [Student("Ana", 8), Student("Luis", 6)]
sorted(students, key=lambda s: s.grade)
# → [Luis(6), Ana(8)]
Résumé syntaxe
sorted(iterable, key=..., reverse=...)
Retourne une NOUVELLE liste triée. L'original n'est pas modifié.
lambda x: x[1]
Fonction anonyme. x est l'élément, x[1] est le critère de tri.
key=lambda x: (a, b)
Tri par tuple = critères multiples. Si a égaux, compare b.
reverse=True
Tri descendant. Par défaut reverse=False (croissant).

12 valid_palindrome
string EASY
▼
Consigne

Retourner True si la string est un palindrome, en ignorant la casse et les caractères non-alphanumériques.

Solution
pythondef valid_palindrome(s: str) -> bool:
    # 1. Garder seulement les alphanum, tout en minuscule
    filtered = "".join(c.lower() for c in s if c.isalnum())
    # 2. Comparer avec l'inverse
    return filtered == filtered[::-1]
Décomposition pas à pas
# "Was it a car or a cat I saw?"
# → isalnum + lower → "wasitacaroracatisaw"
# → [::-1]          → "wasitacaroracatisaw"
# → True ✓

# [::-1] = slice avec step -1 = string inversée
Exemples
"Was it a car or a cat I saw?"→True
"A man, a plan, a canal: Panama"→True
"No lemon, no melon"→True
"tab a cat"→False
13 whisper_lipher (César)
string math MED
▼
Consigne

Chiffrement de César : décaler chaque lettre de shift positions. Préserver majuscules/minuscules. Non-lettres inchangés. Circulaire (après 'z' → 'a').

Solution
pythondef whisper_lipher(text: str, shift: int) -> str:
    result = ""
    for char in text:
        if char.isalpha():
            base = ord('A') if char.isupper() else ord('a')
            shifted = (ord(char) - base + shift) % 26 + base
            result += chr(shifted)
        else:
            result += char
    return result
Formule — décryptage
# Étape par étape avec 'H', shift=3 :
# 1. base  = ord('A') = 65  (car 'H' est upper)
# 2. ord('H') = 72
# 3. 72 - 65 = 7        (position dans l'alphabet: 0='A', 7='H')
# 4. (7 + 3) % 26 = 10  (décalage + modulo circulaire)
# 5. 10 + 65 = 75       (remettre dans la plage ASCII)
# 6. chr(75) = 'K' ✓

# Pour 'x', shift=2 :
# ord('x')=120, base=ord('a')=97
# (120-97+2) % 26 = 25 % 26 = 25
# 25 + 97 = 122 → chr(122) = 'z' ✓

# Pour 'z', shift=2 :
# (25 + 2) % 26 = 1 → 1 + 97 = 98 → 'b' ✓ (circulaire)
Exemples
"Hello, World!", 3→"Khoor, Zruog!"
"abc", 1→"bcd"
"xyz", 2→"zab"
