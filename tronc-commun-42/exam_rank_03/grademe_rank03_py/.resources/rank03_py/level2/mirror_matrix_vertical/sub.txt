Assignment name: mirror_matrix_vertical
Expected files: mirror_matrix_vertical.py
Allowed functions: All built-in
--------------------------------------------------------------------------------

Write a function:

def mirror_matrix_vertical(matrix: list[list[int]]) -> list[list[int]]:

Returns a new matrix mirrored vertically, i.e., the order of rows is reversed.
Rows themselves remain unchanged.
The original matrix must not be modified.

Example:

matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

mirror_matrix_vertical(matrix) ->
[
  [7, 8, 9],
  [4, 5, 6],
  [1, 2, 3]
]
