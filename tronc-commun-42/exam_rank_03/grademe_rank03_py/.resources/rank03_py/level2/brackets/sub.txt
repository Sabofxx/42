Assignment name: brackets
Expected files: brackets.py
Allowed functions: All built-in
--------------------------------------------------------------------------------

Write a function:

def brackets(s: str) -> bool:

Takes a string s and returns True if all brackets in the string are
correctly matched, properly nested, and closed, and False otherwise.

The brackets to check are: (), [], {}

Other characters should be ignored.

Examples:

brackets("()")               -> True
brackets("([{}])")           -> True
brackets("(]")               -> False
brackets("([)")              -> False
brackets("a(b[c]d)")         -> True
brackets("[{adaudna}]")      -> True
brackets("[{adaudna}])")     -> False
brackets("abc{[123(xyz)]}")  -> True
